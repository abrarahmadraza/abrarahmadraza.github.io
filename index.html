Privacy Policies

Secure Test Environment Chrome Extension Privacy Policy

1. Information Collection

Our Chrome extension collects the following information during active test sessions:

  • Candidate name (as entered during test initialization)
  • Test ID (as entered during test initialization)
  • Window and tab switching events
  • Test start and end timestamps

2. Use of Information

The collected information is used solely for:

  • Monitoring test session integrity
  • Providing activity logs to test administrators
  • Ensuring compliance with test rules
  • Preventing unauthorized actions during tests

3. Data Storage

All collected data is:

  • Stored temporarily during active test sessions only
  • Cleared when the test session ends
  • Never persisted beyond the test duration
  • Not shared with any third parties except authorized test administrators

4. Security Measures

We implement appropriate security measures:

  • All data transmission is encrypted using HTTPS
  • Access to collected data is restricted to authorized personnel
  • Regular security audits and updates
  • Immediate data deletion after test completion

5. User Rights

Users have the right to:

  • Be informed when monitoring is active
  • Know what data is being collected
  • Request deletion of their data
  • Opt out of using the extension
